### Git - 配置环境变量

> `此电脑` -> `属性` -> `高级系统设置` -> `环境变量`

```
# 新建环境变量
GIT_PATH -> E:\IT_zhengqing\soft\soft-dev\Git\Git

# 编辑PATH环境变量，新增
%GIT_PATH%\bin
%GIT_PATH%\mingw64\bin
%GIT_PATH%\mingw64\libexec\git-core
```

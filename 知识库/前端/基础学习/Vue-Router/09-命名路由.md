为任何路由提供`name`

`my-vue-router/src/router/index.js`

```js
const routes = [
    { path: "/user/:id*", name: "user", component: User },
];
```

`my-vue-router/src/App.vue`

```vue
<router-link :to="{ name: 'user', params: { name: 'zq' } }">
Go User
</router-link>
```

`my-vue-router/src/views/User.vue`

```vue
<template>
  <div>user {{ $route.params.name }}</div>
</template>
```

# kubectl 常用命令

> 见 https://kubernetes.io/zh-cn/docs/reference/kubectl/generated/kubectl/

```shell
# 打印服务器支持的 API 资源
kubectl api-resources
```

### 命名空间

```shell
# 查看命名空间
kubectl get ns

# 查看默认命名空间`default`下的应用 -- 如果操作不指定命名空间，默认都是放在`default`命名空间下
kubectl get pods
# 查看所有命名空间下的应用
kubectl get pods -A
# 查看指定命名空间`xxx`下的应用
kubectl get pods -n xxx

# 创建命名空间 `zq`
kubectl create ns zq
# 删除 -- 会将其下所有应用删除
kubectl delete ns zq



```
